## My own WebOS

A mini "operating system" made with HTML, CSS, and JavaScript, simulating windows, a desktop, and basic applications.

### Project structure
```bash
/css
  style.css
  buttons.css
  desktop.css
  taskbar.css
  topbar.css
  window.css

/img
  cat.png
  drag.png
  miku.png
  moon.png

/js
  catview.js
  script.js
  welcome.js

index.html
```

### How to run

1. Clone the repository:

```bash
git clone https://github.com/Hallow303/MyownWebOS.git

```

2. Access the project folder:

```bash
cd MyownWebOS
```

3. Run the project:

```bash
Open the file index.html in your browser
```

### How to use
1. Open the desktop
2. Click on the app icons
3. Open windows (Cat Viewer / Welcome)
4. Drag windows around the screen

### Notes
This is a simple WebOS project made with HTML, CSS and JavaScript.
No installation required.
